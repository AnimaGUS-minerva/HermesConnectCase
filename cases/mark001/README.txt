# Summary

Inspired by Noe Ruiz, this is an Orange Pi Zero case with tool-less design in mind right off the printer. Print the case body and snap in your zero. Then, print the lid and have it snap in place on top as well. 

# Print Settings

Printer: Monoprice Select Mini
Rafts: Doesn't Matter
Supports: No
Resolution: 0.2mm
Infill: 15%

Notes: 
Printed with 0.8mm wall thickness. The body prints great without any support just as it is. The lid however needs to be flipped over so that the big flat part sits flat on the bed.
